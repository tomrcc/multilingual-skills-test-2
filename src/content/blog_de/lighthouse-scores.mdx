## Barrierefreiheit

Mit Blick auf Barrierefreiheit entwickelt.

* Semantisches HTML
* „Zum Inhalt springen“-Schaltfläche
* Aussagekräftige Aria-Labels
* Aussagekräftige Alternativtexte für Bilder
* Fokussierbares HTML

## Performance

Enthält Bildoptimierung, verzögertes und sofortiges Laden, minimales JavaScript – alles in einer statischen Astro-Website, für geringe Seitengrößen und schnelle Ladezeiten.

## SEO

Steuern Sie SEO-Werte sowohl seitenweise als auch websiteweit. Felder auf einzelnen Seiten werden für die SEO-Werte der jeweiligen Seite verwendet, und falls sie nicht gesetzt sind, dienen die websiteweiten Werte als Rückfalloption. Enthält alle Felder, die Sie brauchen, um Redakteure zu ermutigen, die Inhalte einzutragen, die Suchmaschinen für ein gutes Ranking erwarten. Performance, Best Practices und Barrierefreiheit spielen dabei ebenfalls eine Rolle.
