Ce site utilise Tailwind CSS v4 avec l'approche de configuration « CSS-first ». La feuille de style principale, `src/styles/main.css`, importe Tailwind avec `@import "tailwindcss"`.

Tailwind v4 utilise `@tailwindcss/vite` comme plugin Vite au lieu de l'ancienne intégration `@astrojs/tailwind`. La configuration se fait en CSS avec `@layer theme` plutôt que dans un fichier de configuration JavaScript.

Différences clés par rapport à Tailwind v3 :

- Aucun fichier `tailwind.config.js` nécessaire
- Utilisation de `@import "tailwindcss"` au lieu des directives `@tailwind`
- Configuration via des propriétés personnalisées CSS
- Détection automatique du contenu (pas de chemins `content`)
- Valeurs d'espacement et de grille dynamiques sans étendre la configuration
