Ce template utilise des fichiers de données pour :

* Alimenter les champs de sélection simple et multiple dans CloudCannon.
  * Couleurs
  * Icônes
  * Tags
* Définir des valeurs valables pour tout le site, comme les paramètres SEO globaux.
  * Titre du site
  * Description
  * Image OG
  * Type de page
* Contrôler les données d'en-tête et de pied de page pour permettre aux éditeurs de gérer la navigation.
  * Logo
  * Couleurs
  * Liens de page
  * Liens vers les réseaux sociaux
  * Droits d'auteur
